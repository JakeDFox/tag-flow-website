How it works

Set a rule.
Tags apply themselves.

Build conditions in a simple UI. TagFlow watches every incoming order and applies tags the moment they arrive.

01
Create a rule

Pick your conditions — cart total, country, product tag, gateway, and more.

02
Order comes in

TagFlow receives the order instantly via Shopify webhook.

03
Rules evaluated

Every active rule is checked against the order in priority order.

04
Tag applied

Matching tags are applied immediately. Every action logged.

Features

Built for merchants
who care about reliability.

Most tagging apps work until they don't. TagFlow shows you exactly what happened and fixes it when something goes wrong.

⚡

Real-time tagging

Tags apply the moment an order is placed via Shopify webhooks — not on a delay, not in batches.

📋

Full audit log

Every rule run is logged — applied, skipped, or failed. See exactly why a tag was or wasn't applied to any order.

🔄

Manual sync

Install TagFlow on an existing store? Run a sync to tag your historical orders against your rules instantly.

🎯

Flexible conditions

Cart total, shipping country, product tags, payment gateway, customer tags, item count, and more.

↕️

Rule priority

Rules run top to bottom. Drag to reorder and control exactly which tags take precedence.
